# Cinema Booking System

## Overview

Cinema Booking System is a C# Windows Forms application developed as part of a Data Structures course. The system simulates a real-world cinema reservation process, allowing users to browse movies, view movie details, select seats, and complete ticket bookings through an interactive graphical interface.

The project applies multiple data structures to efficiently manage movies, seat reservations, and waiting lists while providing a smooth user experience.

## Features

- Browse available movies
- View detailed movie information
- Interactive seat selection
- Reserve cinema tickets
- Waiting list when all seats are booked
- User-friendly Windows Forms interface
- Real-time reservation management

## Technologies Used

- C#
- Windows Forms
- Data Structures

## Data Structures Implemented

### Linked List
Used to manage seat reservations dynamically and efficiently handle booked seats.

### Queue
Used to implement a waiting list system. When all seats are occupied, additional users are placed in a queue and processed in First-In-First-Out (FIFO) order.

### List
Used to store and manage the collection of movies displayed in the application.

## System Workflow

1. Display available movies on the main screen.
2. Select a movie to view its details.
3. Click the booking button to open the seating layout.
4. Choose an available seat.
5. Confirm the reservation.
6. If all seats are occupied, the user is added to the waiting list.

## Learning Objectives

This project was designed to apply fundamental Data Structures concepts in a practical scenario and demonstrate how different structures can be used to solve reservation and queue management problems.
